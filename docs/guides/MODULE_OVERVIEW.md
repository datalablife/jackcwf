      - `error_code` - 错误代码

    - Level 2: 上下文缓存 (ephemeral) - 对话历史、文档上下文
    - Level 3: 查询缓存 (ephemeral) - 常见问题、相同查询
  - **性能特性**：

    - 成本节省: 90% (缓存读取 vs 正常成本)

## 5 基础设施监控模块 (Phase 1 新增 ✨)
- `src/infrastructure/cache_metrics.py`
  - **模块用途**：Prometheus 指标定义与记录，用于语义缓存监控
  - **核心指标定义**（17+ 指标）：
    - **计数器 (Counter)**：

  - **成本计算模型**：

    - 总成本 = 缓存读取成本 + 缓存写入成本 + 正常查询成本
