# 🎯 最简单的修复方法 - 只需点击！

**你的问题**: 用户无法登录，显示 "Cannot connect to server: timeout"

**好消息**: 修复非常简单，只需要点击几下鼠标。预计 5 分钟解决。

---

## ✅ 第一步：打开 Coolify 管理面板

1. **打开你的浏览器**
2. **在地址栏输入**:
   ```
   https://coolpanel.jackcwf.com
   ```
3. **按 Enter 键**，访问管理面板
4. **输入你的 Coolify 登录信息**（用户名和密码）
5. **登录**

---

## ✅ 第二步：找到你的应用

登录后，你应该看到一个仪表板。现在：

1. **在左侧菜单或主页上找到 "Applications"（应用）**
2. **在应用列表中找到你的应用**，通常会显示：
   - 应用名: `datalablife/jackcwf` 或 `jackcwf`
   - 状态: `running:healthy` (绿色)
3. **点击这个应用进入详情页**

---

## ✅ 第三步：重新启动应用

在应用详情页面，你应该看到一些按钮或菜单。现在：

1. **寻找 "Actions"（操作）菜单或按钮** - 通常在页面顶部或右上角
2. **在菜单中找到以下选项之一**:
   - ✅ "Restart"（重新启动）- 最简单
   - ✅ "Redeploy"（重新部署）
   - ✅ "Restart Container"（重启容器）
3. **点击这个按钮**
4. **如果有确认对话框，点击 "Yes" 或 "Confirm"（确认）**

---

## ✅ 第四步：等待应用重启

应用重启时：

- 你会看到状态从 `running:healthy` 变成 `restarting`（重启中）
- **耐心等待 1-2 分钟**
- 状态会变回 `running:healthy` ✓

---

## ✅ 第五步：测试修复

等应用重启完成后：

1. **打开一个新的浏览器标签页**
2. **在地址栏输入**:
   ```
   https://www.jackcwf.com
   ```
3. **按 Enter 键**
4. **观察页面**:
   - ❌ 如果仍然显示超时错误 → 继续下面的 "步骤 6"
   - ✅ 如果页面正常加载，显示登录框 → 修复成功！

---

## ✅ 第六步：验证修复成功

现在验证 WebSocket 连接是否正常：

1. **在刚才访问的页面（https://www.jackcwf.com）上按 F12**
   - 会打开开发者工具窗口
2. **在顶部找到并点击 "Network" 标签**
3. **按 F5 或 Ctrl+R 刷新页面**
4. **在 Network 标签中寻找包含 "_event" 的请求**:
   - 类型应该显示为 "WS"（WebSocket）
   - 状态应该显示为 "101" 或 "101 Switching Protocols"
5. **尝试登录**:
   - 输入用户名
   - 输入密码
   - 点击登录
   - 应该能成功登录 ✓

**成功标志**:
- ✅ Network 中能看到 WebSocket 连接
- ✅ 连接的状态是 "101"
- ✅ 能成功输入用户名和密码
- ✅ 能成功登录
